# Tic-Tac-Toe-C
🎮Tic TAC Toe game in c
📌 **Description**
This is a simple Tic Tac Toe game built using C programming language. It allows two players to play on a 3*3 board
🚀**Features**
. Two-player mode(x and 0)
. Win detection 
. Draw detection
. Input validation
💡**Concepts used**
. 2D arrays
. Loops
. Function
. Conditional statements
